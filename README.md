# hello_conan
This repository contains examples that I used while exploring conan.

## Conan
[Conan](https://conan.io) is an open source, decentralized and multi-platform package manager to create and share all your native binaries.
Official documentation: [https://docs.conan.io/en/latest/](https://docs.conan.io/en/latest/)

## Requirements

* python 3.7 or higher 
* pip 20.0.2 or higher
* conan version 1.26.0 or higher (pip install conan)
* cmake version 3.18.2 or higher (pip install cmake)
* make version 3.81 or higher

## What's covered?

* ex1: Creating a simple hello world conan package.
* ex2: Using an existing conan package as a dependency in your code. 
* ex3: Handling transitive dependencies using conan.
* ex4: Integration with artifactory.
* ex5: Using conan profiles for cross compilation.
* ex6: ABI compatibility in conan packages.
